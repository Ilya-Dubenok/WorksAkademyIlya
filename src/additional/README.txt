-КОЛЛЕКЦИИ-
Задание в пакете additional.collections

пакет collections.dto включает классы Animal и Person
пакет collections.utils включает:
-компараторы (для Person и Animal)
-класс NameGenerator - класс со статическими методами для генерации имен/псевдонимов/кличек различными методами (рандом, из файла, различные алфавиты)
-класс CollectionsUtils - класс со статическими методами для работы над коллекцией (заполнение, замер времени выполнения, сортировка, итерирование, удаление данных)
пакет collections.run включает класс Main, где точка входа в программу и демонстрация работы программы
пакет collections.tests включает класс Tests, где размещены тесты выполнения методов
